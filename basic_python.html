print()

print() is a built-in Python function used to output some text in the terminal.

print("Hello world!")

In this example "Hello world!" is a parameter value (or an argument) passed into the print() function.

Variable

A variable is a piece of computer memory that can store a value. In Python, variables can be initialized as

var1 = 7 # A variable named 'var1' storing an integer of value 7
var2 = 99.1 # A variable named 'var2' storing a float of value 99.1
var3 = "Pizza" # A variable named 'var3' storing a string of value "Pizza"

You can then print the values with print()

print(var1) # output : 7

Identifier

An identifier is a name to identify a Python variable, class or function. For example

# example variable identifier
variable_name = "Hamburger" # This variable's identifier is "variable_name"

# example function identifier
def customFunction(arg1): # This function's identifier is "customFunction"
    return arg1

# example class identifier
class CustomClass: # This class's identifier is "CustomClass"
    def __init__(self, class_arg1, class_arg2):
        self.class_arg1 = class_arg1
        self.class_arg2 = class_arg2
